Font color

Font — контейнер для изменения характеристик шрифта, таких как размер, цвет и гарнитура.

Атрибуты
Size - Задает размер шрифта в условных единицах.
Color - Устанавливает цвет текста.
Face - Определяет гарнитуру шрифта.
  • Заменяем все неправильные шестнадцатеричные символы нулями 0, BruceWillis превращается в b00ce000000
  • “Добиваем” число нулями до количества знаков, делимого на 3 (11 -> 12), получаем b00c e000 0000
  • Разбиваем на три группы, каждая из которых отвечает за одну компоненту RGB: RGB (b00c, e000, 0000)
  • Обрезаем каждый из аргументов справа, оставляя 2 символа, и получаем наш результат RGB (b0, e0, 00) = #B0E000 или он же RGB(176, 224, 0).
